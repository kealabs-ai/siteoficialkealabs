# Imagem de Fundo do Hero

Para adicionar a imagem de fundo da seção Hero (Início):

1. **Baixe uma imagem de tecnologia/dados/BI** de sites como:
   - Unsplash: https://unsplash.com/s/photos/data-technology
   - Pexels: https://www.pexels.com/search/business-intelligence/
   - Pixabay: https://pixabay.com/images/search/technology/

2. **Sugestões de busca:**
   - "data visualization"
   - "business analytics"
   - "technology dashboard"
   - "digital transformation"
   - "artificial intelligence"

3. **Renomeie a imagem para:** `hero-bg.jpg`

4. **Coloque nesta pasta:** `src/assets/hero-bg.jpg`

5. **Especificações recomendadas:**
   - Resolução: 1920x1080 ou maior
   - Formato: JPG ou PNG
   - Tamanho: Otimizado (< 500KB)
   - Tema: Tecnologia, dados, gráficos, dashboards
